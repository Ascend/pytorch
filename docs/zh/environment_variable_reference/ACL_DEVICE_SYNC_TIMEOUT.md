# ACL\_DEVICE\_SYNC\_TIMEOUT

## 功能描述

通过此环境变量可配置设备同步的超时时间。

单位为s，配置范围为[1,2,147,483]，若不配置则为设备默认同步超时时间。

## 配置示例

```
export ACL_DEVICE_SYNC_TIMEOUT=300
```

## 使用约束

无
