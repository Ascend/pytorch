# 接口变更说明

本章节的接口变更说明包括新增、修改、废弃和删除。接口变更只体现代码层面的修改，不包含文档本身在语言、格式、链接等方面的优化改进。

-   新增：表示此次版本新增的接口。
-   修改：表示本接口相比于上个版本有修改。
-   废弃：表示该接口自作出废弃声明的版本起停止演进，且在声明一年后可能被移除。
-   删除：表示该接口在此次版本被移除。
