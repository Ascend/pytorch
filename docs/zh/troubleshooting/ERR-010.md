# ERR\*\*010

该错误码表示系统调用错误。“\*\*“表示不同模块，例如ERR00010代表PTA框架侧存在系统调用错误。

## Symptom

\[%s\] ERR00010 PTA system call failed.

## Possible Cause

System call returns some errors.
