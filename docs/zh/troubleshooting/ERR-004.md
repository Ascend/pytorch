# ERR\*\*004

该错误码表示存在无效指针。“\*\*“表示不同模块，例如ERR01004代表OPS算子侧存在无效指针参数。

## Symptom

\[%s\] ERR01004 OPS invalid pointer.

## Possible Cause

Some pointer variables are not expected.
