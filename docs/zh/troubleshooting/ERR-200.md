# ERR\*\*200

该错误码表示HCCL接口调用错误。“\*\*”表示不同模块，例如ERR02200代表分布式DIST模块调用HCCL接口出现错误。

## Symptom

\[%s\] ERR02200 DIST call hccl api failed.

## Possible Cause

Call HCCL API returns some errors.

## Solution

Check CANN-related errors in logs and find the CANN Error Code.

> [!NOTE]  
> CANN软件相关报错的详细介绍请参见《[CANN 故障处理](https://www.hiascend.com/document/detail/zh/canncommercial/83RC1/maintenref/troubleshooting/troubleshooting_0001.html)》。
