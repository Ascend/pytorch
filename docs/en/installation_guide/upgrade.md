# Upgrade

<!-- md-trans-meta sourceCommit=unknown translatedAt=2026-06-15T03:38:35.284Z pushedAt=2026-06-15T07:27:21.219Z -->

1. You need to manually uninstall the source version first. For details, see [Uninstallation](uninstallation.md).
2. Install the target version. For details, see [Installing PyTorch](installing_PyTorch.md).

> [!NOTE]
>
> - In a matching firmware, driver, and CANN software environment, you can upgrade only the PyTorch framework and the torch_npu plugin.
> - If you also need to upgrade the firmware, driver, and CANN software, upgrade them in the order of "firmware -> driver -> CANN software" first, and then upgrade the PyTorch framework and the torch_npu plugin.
