This error code indicates that a feature is not supported. "\*\*" represents the specific module, for example, ERR00007 indicates that an unsupported framework API was called.

## Symptom

\[%s\] does not support \[%s\]. ERR00007 PTA feature not supported.

## Possible Cause

Some features are not supported.

## Solution

Based on the error information, use alternative similar functions instead, or submit an issue to request support at [https://gitcode.com/Ascend/pytorch/issues](https://gitcode.com/Ascend/pytorch/issues).
