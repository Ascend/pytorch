# ERR\*\*003

<!-- md-trans-meta sourceCommit=unknown translatedAt=2026-06-12T08:21:09.042Z pushedAt=2026-06-12T11:22:41.018Z -->

This error code indicates a value error. "\*\*" represents the specific module, for example, ERR01003 indicates that there is an error in the value, shape, or dimension of a parameter on the OPS operator side.

## Symptom

\[%s\] ERR01003 OPS invalid value.

## Possible Cause

The value of input parameters is not as expected.

## Solution

Check the input's value, shape, dimension and so on.
