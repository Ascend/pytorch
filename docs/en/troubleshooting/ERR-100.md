# ERR\*\*100

<!-- md-trans-meta sourceCommit=unknown translatedAt=2026-06-12T08:22:12.636Z pushedAt=2026-06-12T11:22:41.032Z -->

This error code indicates an ACL API call error. "\*\*" represents the specific module, for example, ERR00100 indicates an error occurred when calling an ACL API on the PTA side.

## Symptom

\[%s\] ERR00100 PTA call acl api failed.

## Possible Cause

Calling the ACL API returns some errors.

## Solution

Check CANN-related errors in logs and find the CANN error code.

> [!NOTE] 
> For details about CANN software-related errors, see [CANN Troubleshooting](https://www.hiascend.com/document/detail/en/CANNCommunityEdition/900/maintenref/troubleshooting/troubleshooting_0001.html).
