# ERR\*\*004

<!-- md-trans-meta sourceCommit=unknown translatedAt=2026-06-12T08:21:06.990Z pushedAt=2026-06-12T11:22:41.017Z -->

This error code indicates that an invalid pointer exists. "\*\*" represents the specific module, for example, ERR01004 indicates that an invalid pointer parameter exists on the operator side.

## Symptom

\[%s\] ERR01004 OPS invalid pointer.

## Possible Cause

Some pointer variables are not expected.
