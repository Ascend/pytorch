# ERR\*\*008

<!-- md-trans-meta sourceCommit=unknown translatedAt=2026-06-12T08:21:33.666Z pushedAt=2026-06-12T11:22:41.024Z -->

This error code indicates that the relevant resource cannot be found. "\*\*" represents the specific module, for example, ERR00008 indicates that the relevant resource cannot be found on the PTA framework side.

## Symptom

\[%s\] ERR00008 PTA resource not found.

## Possible Cause

Some files are not found.

## Solution

1. Check if the execution environment is configured correctly.
2. Check if the required files exist.
