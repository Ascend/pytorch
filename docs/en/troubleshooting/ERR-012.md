# ERR\*\*012

<!-- md-trans-meta sourceCommit=unknown translatedAt=2026-06-12T08:21:59.707Z pushedAt=2026-06-12T11:22:41.028Z -->

This error code indicates a permission error. "\*\*" represents the specific module, for example, ERR00012 indicates a permission error on the PTA side.

## Symptom

\[%s\]. ERR00012 PTA permission error.

## Possible Cause

The permissions of the file or directory have some issues.

## Solution

According to the error information, modify the permissions of the file or directory specified in the error message.
