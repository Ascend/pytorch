# ERR\*\*300

<!-- md-trans-meta sourceCommit=unknown translatedAt=2026-06-12T08:22:22.934Z pushedAt=2026-06-12T11:22:41.033Z -->

This error code indicates a GE API call error. "\*\*" represents the specific module, for example, ERR00300 indicates an error occurred when the PTA side called the GE API.

## Symptom

\[%s\] ERR00300 PTA call ge api failed.

## Possible Cause

Calling the GE API returns some errors.

## Solution

Check CANN-related errors in logs and find the CANN error code.

> [!NOTE]
> For details about CANN software-related errors, see [CANN Troubleshooting](https://www.hiascend.com/document/detail/en/canncommercial/900/maintenref/troubleshooting/troubleshooting_0001.html).
