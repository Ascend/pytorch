# ERR\*\*009

<!-- md-trans-meta sourceCommit=unknown translatedAt=2026-06-12T08:21:35.576Z pushedAt=2026-06-12T11:22:41.025Z -->

This error code indicates that a resource is unavailable. "\*\*" represents the specific module, for example, ERR04009 indicates that a resource is unavailable in the PROF module.

## Symptom

1. \[%s\] ERR04009 PROF resource unavailable.
2. \[%s\] ERR00009 PTA resource unavailable.

## Possible Cause

Some required resources are unavailable.

## Solution

According to the error information, fix the resource problem.
