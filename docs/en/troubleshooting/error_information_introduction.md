# About Error Message Analysis

<!-- md-trans-meta sourceCommit=unknown translatedAt=2026-06-12T08:24:39.646Z pushedAt=2026-06-12T11:22:41.055Z -->

You may encounter various error messages during model training, including Python call stacks and exception information, native framework error messages, torch_npu error codes, CANN software error codes, and plogs. This section provides guidance on analyzing and locating the source of errors.

> [!NOTE] Note
>
> - The main principle for locating errors: focus on the first error message.
> - For detailed information on CANN software-related errors, see [CANN Troubleshooting](https://www.hiascend.com/document/detail/en/canncommercial/900/maintenref/troubleshooting/troubleshooting_0001.html).
