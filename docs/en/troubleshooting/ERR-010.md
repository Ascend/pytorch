# ERR\*\*010

<!-- md-trans-meta sourceCommit=unknown translatedAt=2026-06-12T08:21:45.734Z pushedAt=2026-06-12T11:22:41.026Z -->

This error code indicates a system call error. "\*\*" represents the specific module, for example, ERR00010 indicates a system call error on the PTA framework side.

## Symptom

\[%s\] ERR00010 PTA system call failed.

## Possible Cause

System call returns some errors.
