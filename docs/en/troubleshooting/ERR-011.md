# ERR\*\*011

<!-- md-trans-meta sourceCommit=unknown translatedAt=2026-06-12T08:22:00.355Z pushedAt=2026-06-12T11:22:41.029Z -->

This error code indicates a timeout access. "\*\*" represents the specific module, for example, ERR02011 indicates a timeout access in the distributed (DIST) module.

## Symptom

\[%s\] ERR02011 DIST timeout error.

## Possible Cause

1. Unexpected situations occur during the communication link.

2. The operations on different nodes are inconsistent.

## Solution

1. Find the error line from the log, and fix it.

2. Check if the timeout variable settings are reasonable.
