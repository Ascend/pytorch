# ERR\*\*001

<!-- md-trans-meta sourceCommit=unknown translatedAt=2026-06-12T08:20:42.370Z pushedAt=2026-06-12T11:22:41.012Z -->

This error code indicates the presence of an invalid parameter. "\*\*" represents the specific module, for example, ERR00001, where "00" after "ERR" represents the PTA module, and ERR00001 indicates an invalid parameter on the PTA side.

## Symptom

\[%s\] ERR00001 PTA invalid parameter.

## Possible Cause

The input parameters are invalid.

## Solution

Check if the input parameters are as expected.
