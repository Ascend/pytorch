# 问题定位方法：
参考Pytorch社区文档：《[问题定位方法](https://docs.pytorch.org/docs/stable/user_guide/torch_compiler/torch.compiler_troubleshooting.html)》。
如果出现其他错误，请按问题反馈流程提交ISSUE至Ascend Extension for PyTorch社区。

# 问题反馈流程：
Issue 中应提供 PTA（Ascend Extension for PyTorch）、Triton-Ascend、CANN 的版本信息、最小复现 demo 源码以及报错的堆栈或截图，提交至《[Ascend Extension for PyTorch社区](https://gitcode.com/Ascend/pytorch/issues)》，同时在ISSUE名称中添加[Inductor]标签。
