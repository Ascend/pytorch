## TORCHINDUCTOR_MAX_AUTOTUNE （同社区）

## 功能描述

该环境变量用于控制是否开启max autotune功能，与社区一致。

"0"为关闭，"1"为开启

默认配置为TORCHINDUCTOR_MAX_AUTOTUNE="0"

## 配置示例

开启max autotune功能

```shell
export TORCHINDUCTOR_MAX_AUTOTUNE=1
```

关闭max autotune功能

```shell
export TORCHINDUCTOR_MAX_AUTOTUNE=0
```
